THE VOID by DarkWave Studios

THE VOID is a voice-first mental wellness platform where users record their frustrations — called "venting" — and receive AI-generated responses tailored by five distinct personality modes: Smart-ass, Calming, Therapist, Hype Man, and Roast Master. It's designed to be a cathartic, safe, and engaging space for people to process emotions through voice, text, and guided wellness tools.

Core Experience

Users speak their frustrations into the app. The audio is transcribed in real time using OpenAI's transcription model, then processed by GPT-5.2, which generates a response in the selected personality's voice. Responses can be read as text or played back as speech using OpenAI's text-to-speech with personality-specific voices. Conversations support multi-turn threads, so users can go back and forth with their chosen AI personality. Premium users can fine-tune personality traits like sarcasm and empathy using sliders.

Subscription Model

THE VOID runs on a Stripe-powered subscription with a Founders Pricing model:

First 1,000 subscribers lock in $9.99/month forever (Founders Rate)
After that, new subscribers pay $14.99/month
Credit packs are available for purchasing additional vents beyond the subscription allowance
Founders receive a permanent badge and their rate is preserved for life
Void ID & Blockchain Hallmark

Premium subscribers receive a unique Void ID (format: V-XXXXXXXX) that serves as both a membership hallmark and an affiliate referral code. Each Void ID gets a SHA-256 hash-chained stamp minted on the Trust Layer v1 network — a blockchain-style verification of membership.

Affiliate Program

Premium users share their Void ID as a referral code. When a referred user converts to Premium, the referrer earns 5 bonus credits.

Security & Authentication

Access is secured through a PIN-based whitelist system with master key access, first-login PIN change flow, and Replit Auth (OIDC). The app includes robust rate limiting across all API endpoints and integrates TrustShield.tech for additional security.

Signal Chat

A real-time global chat system with crisis hotline integration, powered by Trust Layer SSO and WebSockets. Includes ecosystem login — members from any DarkWave app (Happy Eats, TrustHome, Signal, Orbit Staffing, etc.) can sign in using their Trust Layer ID or ecosystem email with their password or short numeric PIN.

Trust Layer SSO & Ecosystem

THE VOID is part of the DarkWave Studios ecosystem. All DarkWave apps share a common identity system called Trust Layer SSO. Members get a Trust Layer ID (format: tl-xxxx-xxxx) that works across every app in the ecosystem. This enables cross-app member recognition, shared identity, and ecosystem-wide login.

TrustVault Integration

Premium users' vent audio, voice journals, void echoes, and mood portraits are automatically archived to TrustVault (trustvault.replit.app) with encrypted storage. Non-blocking uploads ensure the core app experience is never interrupted. Free users have their audio transcribed then discarded.
